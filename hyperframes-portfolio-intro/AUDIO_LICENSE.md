# Audio License / Provenance

## Music bed

`assets/original-cinematic-bed.wav` is an original synthesized soundtrack generated locally by `scripts/generate_original_soundtrack.py` for this portfolio intro.

- No third-party music.
- No samples.
- No loops.
- No external recordings.
- Generated from deterministic sine/noise synthesis in Python stdlib.

## Voice-over version

`assets/narration-de-conrad.mp3` was generated from `assets/narration-de.txt` using Microsoft Edge TTS voice `de-DE-ConradNeural`.

- Male German neural TTS voice.
- No voice cloning.
- No imitation of a real person.
- Text written specifically for this portfolio intro.

For strict commercial/public usage, Microsoft service terms for Edge/Azure neural voices should be respected. This is safer than using unofficial VoiceBox/voice-clone models because the voice source is explicit and not cloned from an individual.
