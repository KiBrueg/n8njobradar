# HyperFrames Portfolio Intro — DESIGN

## Style Prompt
Cinematic cosmic operating system: premium dark editorial, warm copper/amber glow, technical but not generic SaaS. The source inspiration is the existing cosmic portfolio, but the video must feel like its own motion-graphics intro rather than a direct website recording.

## Colors
- Void: `#070504` — background
- Ink brown: `#140f0c` — deep panels
- Parchment: `#f0ebe5` — primary text
- Muted sand: `#a79c90` — secondary text
- Copper: `#b5603e` — personal brand accent
- Violet: `#a855f7` — JobRadar planet / AI signal
- Green: `#22c55e` — automation / delivery signal
- Amber: `#f59e0b` — data/security signal

## Typography
- Display: `Fraunces` if available, fallback Georgia/serif
- UI/body: `Inter` if available, fallback system sans
- Numeric/technical accents: system monospace

## Motion Rules
- Slow orbital camera, no chaotic spins.
- Cards enter with blur + scale, pipeline nodes connect with light traces.
- Use cinematic fades and aperture/letterbox framing.
- Avoid jump cuts: every scene crossfades or uses motion continuity.

## What NOT to Do
- No blue SaaS gradients.
- No fake stock dashboard screenshots.
- No tiny unreadable text.
- No random decorative numbers: metrics must come from project context or user correction.
- No direct clone of the previous Three.js site; this is a HyperFrames-native intro.
